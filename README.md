# marketplace-product-analytics

## Overview
This project simulates an Etsy-like marketplace and demonstrates
core product analytics metrics using PostgreSQL.

## Business Goal
Improve retention and repeat purchases.

## Key Metrics

- Total Revenue: 730
- Activation Rate: 80%
- Repeat Rate: 25%
- Average Orders per User: 1.25
- D7 Retention: 0%

## Insights

- Activation is strong (80%), indicating effective onboarding.
- Repeat rate is low (25%), suggesting weak retention.
- No D7 retention observed, meaning users do not return within 7 days.
- Electronics category drives the majority of revenue.

## Tools
- PostgreSQL (Supabase)
- SQL
- Product Metrics Analysis

## Conclusion
The marketplace demonstrates strong initial conversion but weak
retention performance. Strategic focus should shift toward
repeat engagement and habit formation.
